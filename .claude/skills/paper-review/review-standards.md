# Paper Review Standards - 论文评审标准体系

## 概述

本文档定义了适用于不同类型论文的评审标准体系，包括：
- **期刊论文**: PRL / Phys. Rev. A / Nature / Science / Cell 等
- **毕业论文**: 学士 / 硕士 / 博士

---

## 一、评分维度体系

### 1.1 期刊论文评分维度 (100分)

| 维度 | 权重 | 说明 |
|------|------|------|
| **创新性/重要性** | 25% | 对领域的贡献程度 |
| **学术规范性** | 20% | 格式、引用、语言 |
| **理论框架** | 20% | 推导完整性、自洽性 |
| **实验/数据质量** | 15% | 方法严谨性、数据可靠性 |
| **写作质量** | 10% | 逻辑性、可读性 |
| **图表质量** | 10% | 清晰度、规范性 |

### 1.2 毕业论文评分维度 (100分)

| 维度 | 权重 | 说明 |
|------|------|------|
| **研究问题** | 20% | 问题明确性、创新性、价值 |
| **文献综述** | 15% | 覆盖度、批判性分析 |
| **方法论** | 20% | 设计合理性、技术严谨性 |

## 二、期刊论文评审标准

### 2.1 PRL (Physical Review Letters) - 物理快报

#### Desk Rejection 标准 (一票否决)

```
□ 物理重要性: 必须对物理学界广泛影响
□ 创新性: 必须有实质性新物理见解
□ 时效性: 必须解决当前热点问题
□ 长度: ≤4页 或 ≤3750字
□ 格式: 必须使用 revtex4-2
```

#### 评审维度权重

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 物理重要性 | 30% | 必须吸引广泛物理学读者 |
| 创新性 | 25% | 必须有清晰的新物理贡献 |
| 理论完整性 | 20% | 推导必须严谨自洽 |
| 格式合规 | 15% | 严格遵守PRL格式 |
| 写作质量 | 10% | 简洁明了 |

#### 评分细则 (1-5分)

**创新性 (5分制)**
- 5: 开辟新方向/解决长期问题
- 4: 重大突破/显著改进现有理论
- 3: 有价值的进展/新方法
- 2: 增量改进/部分新见解
- 1: 无实质性创新

**理论完整性 (5分制)**
- 5: 完美自洽、完整推导、边界条件清晰
- 4: 基本完整、小幅简化可接受
- 3: 主要框架完整、部分细节不足
- 2: 存在逻辑漏洞或不自洽
- 1: 严重理论错误

**技术严谨性 (5分制)**
- 5: 完美无瑕、可以直接使用
- 4: 有小瑕疵但不影响结论
- 3: 主要方法正确、有改进空间
- 2: 存在技术问题需要修正
- 1: 方法有根本性问题

## 三、毕业论文评审标准

### 3.1 博士论文评审标准

#### 评审维度权重

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 研究问题 | 20% | 原创性问题、有学术价值 |
| 文献综述 | 15% | 全面深入、批判性分析 |
| 方法论 | 20% | 创新性、严谨性 |
| 结果分析 | 20% | 深度、统计分析 |
| 创新贡献 | 15% | 原创贡献、学术贡献 |
| 写作规范 | 10% | GB/T 7714、章节完整 |

#### 评分等级定义

**优秀 (90-100分)**
- 研究问题: 原创性高、有国际领先潜力
- 文献综述: 全面系统、有独到见解
- 方法论: 有重大创新或突破
- 结果: 完整可靠、分析深入
- 创新贡献: 对学科有实质性推动

**良好 (80-89分)**
- 研究问题: 有创新性
- 文献综述: 较全面
- 方法论: 有改进
- 结果: 较完整
- 创新贡献: 有一定贡献

**合格 (70-79分)**
- 研究问题: 明确但创新性一般
- 文献综述: 基本全面
- 方法论: 合理但无创新
- 结果: 完整但分析深度有限
- 创新贡献: 有限但有参考价值

**不合格 (<70分)**
- 存在重大缺失或抄袭

## 四、通用评审检查清单

### 4.1 引言审查

```
□ 问题重要性: 为什么要做这项研究?
□ 文献覆盖: 主要相关工作是否都被引用?
□ 研究空白: 现有研究的不足是否明确?
□ 本文贡献: 创新点是否清晰陈述?
□ 结构: 逻辑是否清晰?
```

### 4.2 理论框架审查

```
□ 假设条件: 是否明确列出?
□ 公式推导: 每一步是否有理有据?
□ 符号定义: 是否清晰无歧义?
□ 边界条件: 是否正确处理?
□ 与前人对比: 是否明确区分?
```

### 4.3 方法论审查

```
□ 实验设计: 是否合理可控?
□ 数据采集: 是否可重复?
□ 统计分析: 方法是否正确?
□ 误差分析: 是否完整?
□ 可重复性: 是否提供足够细节?
```

### 4.4 结果审查

```
□ 数据呈现: 是否清晰准确?
□ 图表质量: 是否符合规范?
□ 统计分析: 是否完整?
□ 与理论对比: 是否充分?
□ 意外结果: 是否讨论?
```

### 4.5 讨论审查

```
□ 结果解释: 是否合理?
□ 与前人对比: 是否充分?
□ 局限性: 是否诚实讨论?
□ 应用前景: 是否有价值?
□ 未来方向: 是否明确?
```

### 4.6 结论审查

```
□ 主要发现: 是否清晰总结?
□ 创新点重申: 是否明确?
□ 局限性: 是否提及?
□ 未来方向: 是否合理?
```

### 4.7 引用审查

```
□ 完整性: 所有引用是否有对应条目?
□ 准确性: 信息是否与原文献一致?
□ 时效性: 是否包含近期文献?
□ 分布: 是否覆盖主要工作?
□ 必要性: 每篇引用是否必要?
```

### 4.8 格式审查

```
□ LaTeX编译: 是否无错误?
□ 字数/页数: 是否符合限制?
□ 参考文献: 格式是否统一?
□ 图表标注: 是否清晰完整?
□ 单位: 是否符合国际标准?
```
