## 关键 Prompt

### 生成摘要
```
基于以下大纲和实验结果，生成 200-300 词的摘要：

[大纲要点]
[实验结果]

要求：
- 包含背景、方法、结果、结论
- 突出创新点
- 避免夸大
```

### 生成创新点声明
```
基于用户实验数据：
[实验数据描述]

识别并陈述本文的核心创新点：
1. 理论创新：...
2. 技术创新：...
3. 应用创新：...

每个创新点需包含：
- 具体发现
- 与已有研究的关键区别
- 证据支持
```

## 注意事项

1. **叙事逻辑**: 构建"已有理论 → 实验验证/挑战 → 新发现"的清晰叙事
2. **用户主导**: 创新点必须来自用户实验，AI 只辅助分析和陈述
3. **保守陈述**: 避免夸大实验结果
4. **格式适配**: 根据目标期刊/学位要求调整结构
