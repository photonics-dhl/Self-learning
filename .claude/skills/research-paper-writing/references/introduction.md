# Introduction Writing Guide

## 结构

1. **研究背景** - 该领域的基本情况和重要性
2. **文献差距** - 前人工作的不足
3. **研究目标** - 本文要解决的具体问题

## 写作要点

### 研究背景

- 从宽泛的领域背景逐步聚焦到具体问题
- 使用"漏斗"结构：领域 → 子领域 → 具体问题
- 引用最重要的工作（3-5篇）

### 文献差距

- 按主题/方法分类讨论前人工作
- 每类工作指出共同局限
- 明确本文要填补的空白

### 研究目标

- 具体、可测试的目标
- 与文献差距对应
- 预告本文的主要贡献

## 反向提纲检查

写出你的引言后：
1. 主题是什么？（一句话）
2. 前人做了什么？有什么不足？
3. 本文要做什么？
4. 本文的主要贡献是什么？

如果这些问题中有任何一个无法回答，引言需要修改。

## 常见问题

- 研究背景过于宽泛
- 文献综述变成罗列而非批判
- 差距陈述模糊
- 目标与差距不对应
- 贡献声称过高

## AI模式警惕

以下表达要避免：
- "近年来，XXX取得了显著进展" → 具体说明进展
- "具有重要的理论和实际意义" → 说明具体价值
- "填补了国际空白" → 说明具体填补了什么
