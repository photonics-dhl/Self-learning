# Paper Review Guide - 审稿人视角自审

## 五维度自我审查

在提交前，按以下五个维度审查论文：

### 1. 贡献度 (Contribution)
- 论文的主要贡献是什么？
- 贡献是原创的吗？
- 贡献的重要性如何？

### 2. 写作清晰度 (Writing Clarity)
- 论文是否清晰易读？
- 每个段落是否有明确的主题？
- 术语使用是否一致？

### 3. 实验强度 (Experimental Strength)
- 实验是否充分验证了论点？
- 是否有对照实验？
- 结果是否可重复？

### 4. 评估完整性 (Evaluation Completeness)
- 是否与最新方法进行了比较？
- 消融实验是否充分？
- 是否考虑了其他评估指标？

### 5. 方法设计合理性 (Method Soundness)
- 方法是否合理？
- 是否有理论支撑？
- 实施细节是否充分？

## 论点-证据地图

对摘要和引言中的每个主要论点，验证：
```
Claim | Evidence | Status (supported/needs evidence)
```

如果有任何论点缺乏证据支持，需要弱化或删除。

## 对抗性审查问题

作为怀疑审稿人，问：
1. 这篇文章的最大优势是什么？
2. 我最可能质疑的点是什么？
3. 实验是否足以支持论文的结论？
4. 作者是否过度声明了贡献？
5. 论文的局限性是否被充分讨论？

## 输出格式

完成审查后，返回：
1. 各维度评分 (1-5)
2. 主要问题列表（按严重程度排序）
3. 建议修改
4. 最终决定：接受/小修/大修/拒绝
