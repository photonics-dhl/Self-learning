# 主题综合分析 Prompt

## 目的
对同一主题下的多篇论文进行深度综合分析，识别技术路线、核心权衡、研究空白。

## 输入
- 主题名称
- 该主题下的论文列表（每篇包含：标题、作者、年份、期刊、引用数、摘要、关键指标）
- 该主题的核心科学问题

## 分析框架

### Step 1: 识别技术路线

对于每篇论文，提取：
- **具体技术方法**: 不是"用了激光"而是"800nm、100fs Ti:Sa激光器通过tilted pulse front excitation"
- **关键参数**: 泵浦波长、脉冲宽度、能量、重复频率等
- **性能指标**: 输出功率/能量、带宽、转换效率

按技术路线分组，识别该主题下的主要方法阵营。

### Step 2: 构建性能对比矩阵

```
技术路线    | 代表工作          | 核心参数         | 主要性能指标    | 优势          | 局限
-----------+------------------+------------------+----------------+--------------+--------
路线A      | 文献X (2015)     | 800nm, 100fs     | 2.5 THz, 400μJ | 高能量输出    | 带宽受限
路线B      | 文献Y (2017)     | 1μm, 200fs       | 5 THz, 50μJ    | 超宽带       | 能量较低
路线C      | 文献Z (2020)     | 两次光束         | 2.36%          | 高效率       | 远程受限
```

### Step 3: 核心权衡分析

回答以下问题：
1. **效率 vs 带宽**: 哪个路线在效率上最优？哪个在带宽上最优？是否存在兼顾的方案？
2. **功率 vs 便捷性**: 高功率方案是否以复杂性为代价？
3. **成熟度 vs 潜力**: 已有成熟方案 vs 新兴有潜力但未成熟的方案

### Step 4: 识别 Gap

基于上述分析，识别文献中尚未被充分研究的问题：

**Gap 类型**:
1. **方法空白**: 某种技术路线尚未被尝试（如"双光束方案在1μm泵浦下未被研究"）
2. **参数空白**: 某个参数区间尚未被探索（如"在泵浦能量>10mJ条件下有机晶体的损伤阈值"）
3. **比较空白**: 两种方法在相同条件下的直接对比缺失
4. **理论空白**: 现象有实验观测但缺乏理论解释

**每个 Gap 必须包含**:
- Gap 描述：[具体缺失]
- 文献证据：[哪篇文献支持这个 Gap 的存在]
- 重要性：[为什么这个 Gap 需要被填补]

### Step 5: 共识与争议

**共识**: 多篇论文验证的相同结论
- 证据：文献X、Y、Z 都支持...

**争议**: 结论不一致的地方
- 文献X 认为...，文献Y 认为...
- 可能的解释：[为什么存在分歧]

## 输出结构

```
### [主题名称]

**核心科学问题**: [该领域最根本的问题]

#### 1. 技术路线分类

| 路线 | 代表工作 | 核心方法 | 关键指标 | 适用场景 |
|------|---------|---------|---------|---------|
| A    | X (2015)| 800nm TPFE | 2.5 THz, 400μJ | 高能量需求 |
| B    | Y (2017)| 1μm filamentation | 5 THz, 50μJ | 超宽带需求 |
| C    | Z (2020)| two-color plasma | 2.36% | 高效率需求 |

#### 2. 核心权衡

- **权衡1**: [矛盾双方]
  - 路线A: [特性] → 导致 [局限]
  - 路线B: [不同特性] → 导致 [不同局限]
  - **无法兼得的原因**: [物理/技术限制]

- **权衡2**: ...

#### 3. 领域共识

- [共识1]: [表述]，证据来自[X, Y, Z]
- [共识2]: [表述]，证据来自[X, Y]

#### 4. 主要争议

- [争议1]: [描述]
  - 文献X 认为: [观点]
  - 文献Y 认为: [观点]
  - 可能解释: [原因]

#### 5. 研究空白

| Gap | 描述 | 文献证据 | 重要性 |
|-----|------|---------|-------|
| G1  | [具体缺失] | [支持论文] | [为什么重要] |
| G2  | [具体缺失] | [支持论文] | [为什么重要] |

#### 6. 未来研究方向

基于上述 Gap，未来研究应重点关注：
1. [方向1]: 如何解决 Gap 1（需要[技术突破]）
2. [方向2]: 如何解决 Gap 2（需要[技术突破]）
```

## 注意事项

1. **数据驱动**: 每个结论都要有具体数据支撑，不能泛泛而谈
2. **跨论文综合**: 不是逐篇描述，而是找出模式和趋势
3. **Gap 具体化**: "缺乏研究"不够，要说明"在什么条件下、针对什么问题、有什么数据支撑"
4. **量化比较**: 用具体数字比较性能，不要只用"高/低"等模糊词

## 调用方式
```
Analyze theme [主题名称] with [N] papers.
Core research question: [问题]
Papers: [详细列表，包含摘 要和关键指标]
Compare technical routes, identify gaps, and write thematic synthesis.
```
