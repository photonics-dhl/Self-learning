# Document Skills

文档处理技能集，支持 Word、PDF、PowerPoint、Excel。

## 子技能

| 技能 | 用途 |
|------|------|
| `docx` | Word 文档创建和编辑 |
| `pdf` | PDF 处理和分析 |
| `pptx` | PowerPoint 演示文稿 |
| `xlsx` | Excel 电子表格 |

## 通用原则

1. **字体**: 使用专业字体 (Arial, Times New Roman)
2. **公式**: Excel 必须零公式错误
3. **模板**: 编辑现有文件时保留原有格式
4. **验证**: 创建后必须验证

## 触发场景

- "创建一个 Word 文档"
- "编辑 PDF"
- "生成 PPT"
- "处理 Excel 数据"
- "将论文转换为 Word"
