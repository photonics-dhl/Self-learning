# Beautiful Notes 技能

## 概述

这个技能提供 Obsidian 学术笔记的**规划-生成-审查**三代理全流程，确保每篇笔记在知识树中有独特位置、结构清晰、排版精美。

## 核心流程（三代理）

```
用户请求 → 规划代理(note-planner) → 写作代理(note-generator) → 审查代理(note-reviewer) → 保存
```

| 代理 | 文件 | 职责 |
|------|------|------|
| **规划代理** | `note-planner.md` | 扫描目录，分析关系，输出规划卡 |
| **写作代理** | `note-generator.md` | 根据规划卡生成笔记 |
| **审查代理** | `note-reviewer.md` | 审查笔记质量 |

---

## 阶段1: 规划代理 (note-planner)

**强制执行，不跳过**

执行步骤：
1. 扫描同级目录，了解已有笔记
2. 分析关系（父话题/子话题/兄弟话题）
3. 输出规划卡（格式见下方）
4. 检查差异化（父子话题开头必须不同）
5. 规划结论：创建新笔记 / 追加到已有 / 终止

**规划卡模板**：
```markdown
## [笔记名] 规划卡

### 基本信息
- 目标目录: [目录路径]
- 笔记类型: [concept/paper/method]

### 关系分析
- 父话题: [XXX] — 开头: "[父话题开头]"
- 子话题: [XXX]
- 兄弟话题: [XXX] — 差异是...

### 关系判定
[DUPLICATE / SUBSUMED / CONTAINS / OVERLAPS / INDEPENDENT]

### 核心定位
[一句话说明在知识树中的位置]

### 必须回答的核心问题
1. [问题1]
2. [问题2]

### 禁止重复的内容
[引导到对应笔记的内容]

### 开头差异化
- 父话题开头: "[父话题开头]"
- 本笔记开头: "[新开头方向]" — 与父话题不同 ✓

### 规划结论
[创建/追加/终止 + 原因]
```

---

## 阶段2: 写作代理 (note-generator)

**根据规划卡生成笔记**

- 角色: 光学领域专家 + 学术笔记专家
- 输入: 规划卡 + 主题
- 输出: 完整笔记初稿

**写作原则**：
- 严格遵循规划卡中的"核心定位"和"禁止重复的内容"
- 使用 knowledge-structure 技能的脉络写作法
- 使用 knowledge-planning 技能的差异化原则

---

## 阶段3: 审查代理 (note-reviewer)

**审查笔记质量**

- 角色: Obsidian 学术笔记排版专家
- 检查项: 规划卡遵循 / 脉络清晰 / 排版规范 / 无重复内容

**审查输出模板**：
```markdown
## 笔记审查报告

### 基本信息
- 文件: [文件名]
- 状态: ✅ 通过 / ⚠️ 需修改 / ❌ 严重问题

### 规划卡遵循检查
- [ ] 核心定位正确
- [ ] 禁止重复内容已排除
- [ ] 开头与父话题不同

### 脉络检查
- [ ] 每节有明确目的
- [ ] 因果链清晰

### 排版检查
- [ ] Callout 使用正确
- [ ] 表格/公式/Mermaid 规范
```

---

## 使用方法

### 生成新笔记
当用户说 "生成 XX 笔记" 时：
1. **规划阶段**: 调用规划代理 → 输出规划卡
2. **写作阶段**: 调用写作代理 → 生成初稿
3. **审查阶段**: 调用审查代理 → 审查报告
4. 根据反馈修订（最多3次循环）
5. 保存最终版本

### 审查现有笔记
当用户说 "审查 XX 笔记" 时：
1. 读取笔记内容
2. 对照规划卡检查差异化
3. 对照排版规范逐项检查
4. 输出审查报告和修改建议

### 优化排版
当用户说 "优化笔记排版" 时：
1. 分析当前笔记的排版问题
2. 按照规范进行优化
3. 提交审查

---

## 排版核心原则

| 元素 | 规范 |
|------|------|
| **加粗** | 关键概念、核心参数 |
| *斜体* | 英文术语、变量名 |
| ==高亮== | 需要醒目但非关键 |
| Callout | 物理图像→concept, 公式→formula |
| 表格 | 表头加粗，有单位 |
| 标题 | H1→H2→H3 连续，无跳级 |

---

## 质量门槛

笔记必须满足：
1. 核心概念解释准确
2. 公式正确且完整
3. 层级结构清晰 (H1→H2→H3)
4. 排版符合规范
5. 引用文献存在
6. **规划卡遵循**：核心定位正确，无重复内容
